¿Qué es el Gestor de Contenido?

El CMS integrado te permite crear bloques de contenido reutilizable que puedes incrustar en tus lecciones. Crea componentes HTML, elementos Bootstrap, contenido Markdown o estructuras JSON que se pueden reutilizar en múltiples cursos.
