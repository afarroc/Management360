### Excepciones al propietario estándar

|-----|--------|-------------------|-----------|
| `courses` | `Course` | `tutor` | NO `created_by` |

| `chat` | `Conversation`, `CommandLog`, `AssistantConfiguration` | `user` | NO `created_by` |

| `memento` | `MementoConfig` | `user` | NO `created_by` |

### Sistemas de tiempo real coexistentes

```
Django Channels  → app chat (WebSocket, consumers.py)
Centrifugo       → app rooms (CentrifugoMixin, Outbox, CDC)
```
No son intercambiables. Cada app usa el suyo.
